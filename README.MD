# Open Source Chrome Extensions

Welcome to this repository! This repository includes a collection of customized Chrome extensions that are entirely **free to use for everyone**. 

Currently, the repository contains:
- **[Upwork Keyword Finder + Job Scraper](./upwork-keyword-finder):** A handy extension to analyze keywords in Upwork jobs and easily scrape job data.
- **[Personal Assistant](./personal-assistant):** Your daily productivity companion — features a full-fledged notes keeper, scheduled task reminders, and a right-click web clipper wrapped securely in Chrome's native Side Panel.

## 🚀 Installation Guide

Since these extensions are available directly from the source code, you can easily install them as "Unpacked Extensions" in your Google Chrome browser. Follow these simple steps:

1. **Clone the repository** to your local machine (or download it as a ZIP file and extract it):
   ```bash
   git clone https://github.com/Rao-Abrar-Ahmad/chrome-extensions.git
   ```
2. Open Google Chrome and navigate to the Extensions Management page. You can do this by typing `chrome://extensions/` in the address bar and hitting Enter.
3. Enable **Developer mode** by toggling the switch in the top right corner of the page.
4. Click on the **Load unpacked** button that appears in the top left area.
5. Go to the cloned directory, select the specific extension folder you want to use (for example, the `upwork-keyword-finder` folder), and click "Select Folder".

The extension is now installed locally and ready to use! Make sure to pin it from your extensions puzzle icon.
